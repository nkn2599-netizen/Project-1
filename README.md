# e-plantShopping 🌿

e-plantShopping is a simple React-based e-commerce web application for selling houseplants online.  
It allows users to browse plants, add items to cart, manage quantities, and view total price.

## 🚀 Features

- Landing page with company branding
- Product listing with plant categories
- Add to Cart functionality using Redux
- Cart management (increase, decrease, delete items)
- Cart total calculation
- Simple checkout simulation

## 🛠️ Tech Stack

- React.js
- Redux Toolkit
- JavaScript (ES6)
- CSS

## 📦 Project Structure

- App.jsx → Landing page
- ProductList.jsx → Products page
- Cart.jsx → Cart page
- CartSlice.jsx → Redux state management
